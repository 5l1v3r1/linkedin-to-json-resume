LinkedIn to Json Résumé

This is a small website that exports your LinkedIn profile to a JSON Résumé.

Due to recent restrictions added to the LinkedIn API, this web app can't ask for access to user's full profile. Thus, the exported information is a subset of your LinkedIn profile data.

The source code is available on GitHub.
